# Agentic AI Assistant
This is an Agentic AI project built using Python,LangGraph,RAG(ChromaDB) and Streamlit.
## Features
- Chat-based UI using Streamlit
- Agentic workflow using LangGraph
- Intent detection
- Retrieval-Augmented Generation (RAG) using ChromaDB
- Tool-based response generation
- Session memory(name+question history)
- Policy guardrails
- Human-in-the-loop approval for high-risk queries
- Confidence level for responses
## Tech Stack
- Python
- Streamlit
- LangGraph
- LangChain
- ChromaDB
- HuggingFace Transformers
## Summary
This project demonstrates a complete Agentic AI system with chat interaction,memory,RAG using ChromaDB,tool execution,policy guardrails and human-in-the-loop approval.
It shows how modern AI assistants reason,retrieve knowledge,and safely respond to users using an agent-based workflow.
